# LUMINA — Neural-Adaptive Spatial Interface

> *Technology that adapts to humans.*

LUMINA is a futuristic, biometric-aware OS concept UI — a fully interactive HTML/CSS/JS prototype of a neural-adaptive spatial interface designed for wearables and AR environments in 2035.

---

## 🌌 Live Preview

> Open `index.html` directly in your browser — no server or build step required.

---

## ✨ Features

- **Onboarding Flow** — Device pairing, neural profile sync, and AR spatial calibration
- **Calibration HUD** — Animated neural waveform, heart rate bars, and eye-tracking grid
- **Dashboard** — Real-time biometric metrics (Focus, Stress, Mental Load, Energy)
- **Focus Mode** — Deep-work timer with cognitive clarity tracking
- **Creative Mode** — Idea canvas with AI suggestion engine
- **Recovery Mode** — Guided breathing with session countdown
- **Notifications** — Cognitive cost–ranked alert system
- **Settings** — Neural preferences, biometric permissions, privacy-first design

## 🔒 Privacy by Design

All biometric processing in this concept is local-only. No data leaves the device. LUMINA is designed around the principle of **zero data collection** with end-to-end encryption as a baseline.

---

## 🛠 Customisation

All design tokens live in `:root` CSS variables at the top of `index.html`. You can retheme the entire app by editing:

```css
:root {
  --purple: #7c5cfc;
  --cyan: #3de8c8;
  --pink: #f04f7a;
  /* ... */
}
```
